SkimRoute finds useful sections in long pages, AI chats, and PDFs.

Use Alt+J for the best part and Alt+N for the next important section. PDF Mode improves PDF jumps; local PDFs may need file URL access, and scanned PDFs can use OCR.

Enable local PDF access

Chrome currently prevents SkimRoute from reading PDFs stored on your computer.

  1. Select Open extension settings.
  2. Turn on Allow access to file URLs.
  3. Return to this PDF and select Check again.
