Local OCR navigation research note

Abstract

Abstract. This study evaluates local OCR accuracy, section extraction, and navigation latency for long documents. It summarizes the claim that browser-local processing can preserve privacy while still creating useful maps.

Methods

Methods. We sampled documents, controlled device class, measured extraction latency, and compared selectable text with scanned images. Methods are important context but less useful than the measured findings.

Results

Results. The experiment found that selectable documents reached a ready map quickly, while scanned documents needed explicit OCR startup and clear progress states. The strongest finding is that optional runtimes should initialize once, on demand, and never load on ordinary articles or AI chat pages.

Results also show that exact highlights are reliable when geometry exists, while approximate navigation should remain honest for sources without precise coordinates.

Conclusion

Conclusion. These findings support a split runtime with local-only PDF and OCR processing, deterministic diagnostics, and regression tests that cover page type, navigation, scroll, and highlight behavior.
